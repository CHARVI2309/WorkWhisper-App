# ✅ TaskFlow App

A clean and responsive **Todo / Task Manager** web application built with HTML, CSS, and JavaScript — featuring user authentication with login and signup pages.

---

## 🚀 Features

- 📝 Add, edit, and delete tasks
- ✅ Mark tasks as complete / incomplete
- 🔐 Login and Signup pages with user authentication
- 💾 Data saved in localStorage (no backend needed)
- 📱 Fully responsive design
- 🎨 Clean and modern UI

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure |
| CSS3 | Styling & Layout |
| JavaScript | Logic & Interactivity |
| localStorage | Data Persistence |

---

## 📁 Project Structure

```
TaskFlow-App/
├── index.html       # Main todo app
├── login.html       # Login page
├── signup.html      # Signup page
├── style.css        # Styles
└── script.js        # App logic
```

---

## ⚙️ How to Run

1. Clone the repository:
```bash
git clone https://github.com/CHARVI2309/TaskFlow-App.git
```

2. Open the project folder:
```bash
cd TaskFlow-App
```

3. Open `index.html` in your browser — that's it! No installation needed.
